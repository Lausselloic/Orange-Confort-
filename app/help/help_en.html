Help about the Confort+

Confort+ enables you to adjust display and behavior settings on web pages you are reading.

settings applied to the site

typography

« font size »

This setting allows you to choose font size.

« space between words »

This setting allows you to add space between texts words.

« space between characters »

If characters are too close, increase this value to add more readability.

« space between lines »

If lines are to close, increase this value to facilitate the reading.

« font face »

Allow to modify the defined web site font face to improve the reading confort. There are two available font faces : « Arial » and « Open Dyslexic ».

« text case »

Allow to modify texts display according to your needs: by capitalizing the first letter of each word, to lowercase or to uppercase all text.

layout

« cancel layout »

Replace site fonts with your default fonts (those defined in your browser). Moreover, content is linearized and displayed without columns.

« align text to the left »

Justified texts disturb often reading. This option allows to disable text justification and to align all texts to the left.

« list elements numbering »

Display a numerical index in front of all list elements.

« navigation links appearance »

Personalize site navigation links color and layout. A different setting is applied according to the links status (default links, selected or activated link and visited links).

« display a reading ruler »

Enable the display of a horizontal and/or vertical reading ruler that follows the mouse cursor to help reading. You can choose colors and thickness thereof.

« cancel transparency effects »

Enable deactivation of possible transparency effects of the page. This minimizes disruption when reading content.

« cancel background images »

Remove background images to keep only important informations of the foreground.

« cancel foreground images »

Remove foreground images of the page that could disturb reading. They are replaced by their text alternatives. A link allows to view the image on demand.

colors

Allow to define a font color and a background color.

You can use either a combination of colors (« preset colors combination ») or redefine the setting (« personalized colors combination »).

To go back to the original colors of the browser, you can either Cancel all settings or choose « preset colors combination » inside colors settings.

behaviour

« directly jump to content whenever it’s possible »

This setting allows an automatic positioning on main page content, in particular, by jumping navigation links.

If the webmaster has not clearly marked up in the page code where the content is, this doesn’t work.

« navigation on hover »

Display a remote control with up and down arrows to scroll the page without click and click for you after a while (the delay you have defined).

« automatic selection of elements »

Move through all page clickable elements as all form elements. You just have to type a key to stop it and show up a menu enabling the equivalent of a click or to resume the automatic move.

Confort+ Service settings

« deactivate Confort+ on this site »

This allows not to apply Confort+ Service features if they interfere with the navigation of the site visited.

« hide toolbar »

Hide the general Confort+ service toolbar, but features are still applied to the site, if the service isn’t deactivated for this one. Using “Confort+” link activates this toolbar.

« language change »

Define the Confort+ service toolbar language, this doesn’t modify the language of the visited site.

« delete all settings »

This command puts everything back into its original state, by exhaustively deleting all the settings you could have enabled in Confort+.

compatibility with Assistive Technologies

If you're using assistive technologies, please read this section.

screen readers

If your screen reader is in “mouse cursor” mode, a simple mouse hover can trigger a link if you chose a motor help like “navigation on hover”.

The “automatic selection of elements” mode (looping through all clickable elements in the page) is very difficult to use with a screen reader, we don't advise it.

screen magnifiers

If you have configured your screen magnifier to alter colors in web pages, then the settings of Confort+ may be perceived incorrectly; or they may conflict with your own settings. If so, choose a combination like “keep the original colors”.

voice control

We have found, at least with Dragon Naturally Speaking, that motor help modes are problematic. We advise you not to use them in conjunction with this software.

troubleshooting

cookies

The Confort+ service uses cookies, which are small text files that enable your browser to store your settings on your computer. If your browser doesn't accept cookies, then the service will not operate properly.

javascript

The Confort+ service is a library distributed on the Orange Group websites, and it relies on Javascript code that modifies pages once they are loaded. If for one reason or another you have disabled Javascript (for example because of a technical restriction by your administrator), then the service will not operate properly.

privacy

No data pertaining to your use of Confort+ is stored on our servers. All your settings are stored, for persistence purposes only, under the form of cookies, that you can destroy at any time by using the “Delete all settings” command. Please note, though, that for technical reasons the Confort+ service will only operate properly if cookies are enabled.
